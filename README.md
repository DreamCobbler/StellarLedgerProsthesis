# Stellar Ledger Prosthesis (*1.0.0*)

**Stellar Ledger Prosthesis** is a standalone tool for [Stellaris](https://store.steampowered.com/app/281990/Stellaris/) - a 4X video game made by Paradox Interactive. It allows you to read saved game states and display some information about each empire/star system/planet/etc. present in the game's universe.

Tested on Windows 10.
